# MailKite examples — Python

Runnable, copy-pasteable examples. Each file's header comment lists what to set and `pip install`.

| File | What it shows |
| --- | --- |
| [`01_send_email.py`](01_send_email.py) | Send an email over a verified domain |
| [`02_receive_webhook.py`](02_receive_webhook.py) | Receive inbound mail as a webhook and **verify the HMAC signature** |
| [`03_agent_email_reply.py`](03_agent_email_reply.py) | **AI email agent** — inbound email → Claude drafts a reply → MailKite sends it, threaded |
| [`04_give_your_agent_an_inbox.py`](04_give_your_agent_an_inbox.py) | Give your agent its own address with MailKite's **built-in inbox agent** (no server) |
| [`05_server_login.py`](05_server_login.py) | **Server-side login + register** — your own account, or your users' accounts via OAuth |

Full docs: <https://mailkite.dev/docs> · AI agents: <https://mailkite.dev/docs/ai-agents>
